First-time setup

  1. Install iTunes or Apple Mobile Device Support (for USB driver)
  2. Run npm run setup in the project folder to download tools
  3. Trust your PC on the iPhone when prompted
  4. Enable Developer Mode on iOS 16+ (Settings → Privacy & Security)

Sideloadly is closed-source. This app uses the open-source Sideloader + zsign stack instead — same signing approach.
